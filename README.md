# PO_O_Problema_Do_Poeta

# Introdução
Esse projeto tem como objetivo consolidar a resolução de um problema linear aplicado ao Manejo Florestal conhecido como "O Problema do Poeta", proposto pelo professor Júlio Arce (UFPR) em sua apostila de Programação Linear.

A ideia aqui é utilizar ferramentas de Python para modelar e resolver o problema, e conforme o avanço, criar novas complexidades em cima da mesma estrutura original.

Assim sendo, esse projeto visa se tornar uma fonte didática para eventuais entusiastas da Pesquisa Operacional aplicada à Engenharia Florestal.
