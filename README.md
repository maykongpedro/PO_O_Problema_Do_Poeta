# PO_O_Problema_Do_Poeta

# Introdução
Esse projeto tem como objetivo consolidar a resolução de um problema linear aplicado ao Manejo Florestal conhecido como "O Problema do Poeta", proposto pelo professor Júlio Arce (UFPR) em sua apostila de Programação Linear para Fins Florestais.

A ideia aqui é utilizar ferramentas de Python para modelar e resolver o problema, e conforme o avanço, criar novas complexidades em cima da mesma estrutura original.

Assim sendo, esse projeto visa se tornar uma fonte didática para eventuais entusiastas da Pesquisa Operacional aplicada à Engenharia Florestal.


# Requisitos e ferramentas utilizadas

## Solver
O solver utilizado nos exercícios será o GPLK (gratuito). Para obtê-lo, basta realizar o download no site original: [GLPK](https://www.gnu.org/software/glpk/).

Também existem formas de baixar diretamente pelo terminal ou utilizando gerenciadores de pacotes como o `conda`, porém, não iremos abordar isso nessa README.

## Python
Esse projeto foi criado utilizando a ferramenta de gerenciamento de pacotes e versões chamada [UV](https://docs.astral.sh/uv/guides/install-python/). Assim é garantida a reprodutibilidade dos exercícios, basta seguir os passos de ativação do projeto ao baixar em sua máquina.

## Quarto
Os relatórios com os enunciados e resoluções dos exercícios foram feitos em QUARTO, uma ferramenta para criar documentos, relatórios, apresentações, etc. Optei por ela ao invés de Jupyter Notebooks pela facilidade que temos de transitar entre diferentes códigos e extensões.

Para poder renderizar os documentos no computador, é necesário instalar o software: [Quarto Download ](https://quarto.org/docs/download/)
